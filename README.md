# SplitKar
> Split expenses with friends & groups — effortlessly.

## Tech Stack
- Node.js, Express, EJS
- MongoDB (Mongoose)
- Session-based auth (express-session + connect-mongo)
- Nodemailer + Gmail SMTP
- Deployed on Render

## Setup
1. Clone the repo
2. `npm install`
3. Copy `.env.example` to `.env` and fill values
4. `npm run dev`
